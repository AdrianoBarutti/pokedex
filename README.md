# Pokédex

Projeto em React com o objetivo de construir uma Pokédex utilizando a PokeAPI.

## Status

Projeto em fase inicial de desenvolvimento.

## Objetivo

Criar uma aplicação para visualizar Pokémon da primeira geração, com funcionalidades de busca, filtro por tipo e exibição de detalhes individuais.

## Tecnologias

- React
- JavaScript
- PokeAPI

## Estrutura inicial

O projeto ainda está na fase de configuração base.

## Roadmap

- Implementar mini game "Quem é esse Pokémon"
- Adicionar busca por nome e filtro por tipo
- Otimizar carregamento de dados da API
- Criar modal com abas separadas (Descrição, Stats e Evoluções)
- Substituir imagens padrão por assets personalizados
